Detailed Description

Particle emitter which emits particles randomly from points inside an ellipsoid.

Remarks
This basic particle emitter emits particles from a ellipsoid area. The initial direction of these particles can either be a single direction (i.e. a line), a random scattering inside a cone, or a random scattering in all directions, depending the 'angle' parameter, which is the angle across which to scatter the particles either side of the base direction of the emitter.
